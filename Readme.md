# Vue-IMG-SHOW

这是一个基于 **Vue.js** 的图像展示应用，支持展示图像并进行基本的交互。

## 项目简介

**Vue-IMG-SHOW** 是一个简单的 Vue.js 项目，旨在展示图片，支持基本的图像展示和管理功能。该项目实现了一个响应式界面，适合用于个人图片展示、图库管理等应用场景。

## 功能特点

- 支持展示图片
- 响应式布局，适用于不同设备
- 提供基本的图片交互功能

## 安装步骤

### 1. 克隆项目

首先，克隆这个仓库到本地：

```bash
git clone https://github.com/Miracle-Shen/VUE-IMG-SHOW.git
```
## 安装依赖

进入项目目录并安装依赖：

```bash
cd VUE-IMG-SHOW
npm install
```

启动开发服务器：
```bash
npm run serve
```
